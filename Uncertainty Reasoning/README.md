# 🔬 Uncertainty Reasoning

This component handles probabilistic reasoning under uncertainty using Bayesian networks — a key challenge in real-world clinical decision support.

---

## ⚙️ Requirements

- **Python** 3.8 or newer
- The following Python packages:

| Package | Version |
|---------|---------|
| pgmpy | 1.1.2 |
| matplotlib | latest |
| numpy | latest |

---

## 📦 Installation

If the packages are not installed, run:

```bash
pip install pgmpy==1.1.2 matplotlib numpy
```

---

## ▶️ How to Run

```bash
cd "Uncertainty Reasoning"
python main.py
```
