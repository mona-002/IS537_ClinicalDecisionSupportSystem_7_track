## 📌 Project Overview

This project implements a **Clinical Decision Support System (CDSS)** using core Artificial Intelligence techniques. The system is designed to assist healthcare professionals in making informed clinical decisions by applying AI-based reasoning and problem-solving methods to medical scenarios.

The project is divided into three main AI components, each stored in its own folder:

## 🧠 Components

### 1. Problem Solving & Search
Implements classical AI search strategies to navigate the clinical decision space. Includes algorithms such as BFS, DFS, A*, and heuristic-based search to identify optimal diagnostic or treatment paths.

### 2. Symbolic Reasoning (Prolog)
Uses **Prolog** to represent medical knowledge as facts and rules. The system applies logical inference to derive diagnoses, recommend treatments, and check for drug interactions based on patient symptoms and history.

### 3. Uncertainty Reasoning
Applies probabilistic reasoning techniques (such as Bayesian networks or certainty factors) to handle incomplete or uncertain patient data — a common challenge in real-world clinical environments.
